API access is a privilege — use it responsibly.
The SmartOLT API should not be treated as a live database; systems are expected to cache reusable data and avoid unnecessary repeated requests.
SmartOLT API documentation can be found at https://api.smartolt.com

API method limits

Default API call budget: 1000 calls per hour, 10 calls per second; bursts above 15 calls per second are blocked.

Method name Max calls per hour
get_all_onus_details 15
get_onu_signal 500
get_all_onus_gps_coordinates 5
get_onu_gps_coordinates 500
get_onu_full_status_info 300
get_onus_authorizations_report 5
get_all_onus_details_per_olt 15

Heavy OLT-detail endpoints have a separate budget: 30 calls every 10 minutes per OLT — additional calls are blocked:

  • get_olt_cards_details
  • get_olt_pon_ports_details
  • get_olt_uplink_ports_details
